# Desafio1_MartinQ
Análisis de ventas y rendimiento de 4 tiendas, se realizara basándose en cinco aspectos, facturación total de cada tienda, cuáles son las categorías más populares de cada tienda, cuál es el promedio de evaluación de cada cliente; los productos más y menos vendidos y cuál es el costo promedio del envío desde la tienda hasta el lugar de los clientes.
